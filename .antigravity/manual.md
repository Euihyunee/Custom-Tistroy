# 티스토리 스킨 커스터마이징 및 배포 매뉴얼

이 문서는 `.antigravity` 에이전트가 생성한 커스텀 스킨을 빌드하고, 티스토리 블로그에 적용하는 방법을 설명합니다.

## 1. 사전 준비
- **Node.js**: 최신 버전이 설치되어 있어야 합니다.
- **티스토리 계정**: 관리자 권한이 있는 블로그가 필요합니다.

## 2. 프로젝트 빌드 (Build)
에이전트가 수정한 코드를 실제 티스토리 스킨 파일(HTML, CSS, JS)로 변환하는 과정입니다.

1.  터미널을 열고 프로젝트 루트 경로(`Custom-Tistroy`)로 이동합니다.
    (또는 저장소를 클론합니다: `git clone https://github.com/Euihyunee/Custom-Tistroy.git`)
2.  다음 명령어를 실행하여 빌드합니다.
    ```bash
    npm run build
    ```
3.  명령어가 완료되면 `dist/` 폴더가 생성(또는 갱신)됩니다. 이 폴더 안에 있는 파일들이 최종 결과물입니다.
    -   `index.xml`, `skin.html`, `style.css`
    -   `images/` 폴더 내의 스크립트 및 이미지 파일들

## 3. 티스토리 스킨 적용 방법

### 3.1. 스킨 등록
1.  **[블로그 관리 홈]** -> **[꾸미기]** -> **[스킨 변경]** 메뉴로 이동합니다.
2.  우측 상단의 **[스킨 등록 +]** 버튼을 클릭합니다.
3.  **[저장]** 버튼 옆의 **[추가]** 버튼을 클릭하여 `dist/` 폴더 내의 파일들을 업로드합니다.
    -   **필수 업로드**: `skin.html`, `style.css`, `index.xml`
    -   **Images 폴더**: `images/` 폴더 안에 있는 **모든 파일**을 선택해서 업로드해야 합니다. (Tip: 폴더째로 드래그 앤 드롭이 안 될 경우, 폴더 안에 들어가서 전체 선택 후 드래그하세요.)
4.  파일 목록에 모든 파일이 정상적으로 올라갔는지 확인합니다.
5.  **[저장]** 버튼을 누르고, 스킨 이름을 입력합니다 (예: `Antigravity Skin v1`).

### 3.2. 스킨 적용
1.  **[스킨 변경]** 메뉴의 **[스킨 보관함]** 탭을 클릭합니다.
2.  방금 등록한 스킨(`Antigravity Skin v1`)을 찾아 **[적용]** 버튼을 클릭합니다.

## 4. 에이전트 기능 설정
이 스킨에는 에이전트가 추가한 특별한 설정들이 포함되어 있습니다.

1.  **[꾸미기]** -> **[스킨 편집]** 메뉴로 이동합니다.
2.  우측 설정 패널에서 **[AI 최적화]** 섹션을 찾습니다.
    -   **AI 요약 블록 활성화**: `켜기(ON)`로 설정하면 글 상단에 숨겨진 요약 정보가 생성됩니다.
    -   **SSO/SEO 강화**: `켜기(ON)`로 설정하면 검색 엔진 최적화 메타 태그가 적용됩니다.
3.  **[적용]** 버튼을 눌러 저장합니다.

## 5. 개발자 기능 활용 가이드

### 코드 블록 (Code Block)
Markdown이나 에디터에서 코드 블록을 작성하면 자동으로 라인 넘버와 복사 버튼이 적용됩니다.

### Mermaid 다이어그램
코드 블록의 언어를 `mermaid`로 설정하고 그래프 코드를 작성하면 자동으로 다이어그램으로 렌더링됩니다.
예시:
\`\`\`mermaid
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
\`\`\`
*(백틱 3개 사용)*

### 수식 (LaTeX)
`$$` 로 수식을 감싸면 렌더링됩니다. (기능 구현 예정)

# 마크다운 종합 샘플 테스트

이 문서는 티스토리 스킨 환경에서 마크다운 요소들이 어떻게 보이는지 확인하기 위한 테스트 페이지입니다.

## 1. 커스텀 박스 (Callout) 테스트

새롭게 추가된 **인용문 기반 커스텀 알림창** 기능입니다.
기존처럼 무거운 `HTML div` 태그를 쓰지 않고, 마크다운 인용문(`>`) 바로 아래 줄에 `{: .색상 }` 특수 구문을 적어주면 자동으로 예쁜 박스로 변환됩니다.

### 사용 예시:

> **성공!** 정상적으로 커스텀 박스 스타일이 적용되었습니다!
> 중요한 공지나 팁을 작성할 때 유용하게 사용할 수 있습니다.
> {: .green }

> **주의!** 이 항목은 경고를 나타내는 노란색 박스입니다.
> {: .yellow }

> **참고!** 이 항목은 정보나 안내를 나타내는 파란색 박스입니다.
> {: .blue }

> **위험!** 이 항목은 에러나 치명적 문제를 나타내는 빨간색 박스입니다.
> {: .red }

> **기본 인용문!** 아무 꼬리표(`{: .class }`)를 붙이지 않으면 일반적인 검정 텍스트의 따옴표 인용문이 나옵니다.

## 2. 텍스트 강조 (Text Formatting)

일반적인 텍스트 강조 방식입니다.

- **굵게(Bold)** 처리된 텍스트입니다.
- _기울임(Italic)_ 처리된 텍스트입니다.
- ~~취소선(Strikethrough)~~이 적용된 텍스트입니다.
- 특정 단어를 강조할 때는 `인라인 코드(Inline Code)`를 사용합니다.

## 3. 목록 (Lists)

### 3.1. 순서가 없는 목록 (Unordered List)

- 사과
- 바나나
  - 바나나우유
  - 바나나칩
- 포도

### 3.2. 순서가 있는 목록 (Ordered List)

1. 첫 번째 항목입니다.
2. 두 번째 항목입니다.
3. 세 번째 항목입니다.
   1. 세부 항목 1
   2. 세부 항목 2

## 4. 인용문 (Blockquote)

다른 사람의 말이나 글을 인용할 때 사용합니다.

> "삶이 있는 한 희망은 있다." - 마르쿠스 툴리우스 키케로
>
> > 인용문 안쪽에 중첩해서 인용문을 작성할 수도 있습니다.

## 5. 코드 블록 (Code Block)

개발 관련 글을 작성할 때 필수로 사용하는 코드 블록 문법입니다.

```javascript
// 자바스크립트 샘플 코드
function greet(name) {
  console.log(`안녕하세요, ${name}님!`);
  return true;
}

const user = "Tistory";
greet(user);
```
